# Usage
1. Download the .ZIP [here](https://github.com/Debugofthe67/portablebrowser-html/releases/download/release/portablebrowse.zip)
2.  Unzip the Compressed FIle
3.  Click on applicationbrowser.exe
4.  Wait for a few seconds (might take a while because its a html app)
